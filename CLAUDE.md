# punkgame — Game01

Гра (ActionScript 3 / FlashDevelop-проєкт).

## Що це
- Проєкт **ActionScript 3**: `punkgame.as3proj`, код у `src/`, бібліотеки в `lib/`, збірка в `bin/` та `obj/`.
- Веб-версія: `web/`, `index.html`.
- `tilegrid.oep` — мапа/рівень (OGMO Editor).

## Межі проєкту
Працювати ТІЛЬКИ в цій папці (`TEST\punkgame`). Інші проєкти в `TEST` не чіпати.

## Git
- Репозиторій: **vh-ss/Game01**, гілка `main`.
- Після кожної зміни — самостійно `commit` + `push`.
- Push працює лише з `http.sslBackend=schannel` (виставлено глобально).
- `bin/`, `obj/` — артефакти збірки; стежити, щоб були в `.gitignore`.
